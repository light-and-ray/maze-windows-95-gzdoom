## Maze Windows 95 UZDoom/GZDoom

**Aka Maze 95 UZDX**
- Trailer: https://www.youtube.com/watch?v=1KAu_HsxvLs
- ModDB: https://www.moddb.com/mods/maze-95-uzdx/downloads/maze-95-uzdx-release
- Or you can clone/download this repository, and pack `maze95` directory into a .zip archive, change extension to `.ipk3`, and it will be the game

Game's [Readme.txt is here](/maze95/Readme.txt)

------------------------
This project is a recreation of Maze Windows 95 screensaver in UZDoom/GZDoom engine with all the features of the original screensaver, including maze generation, walls animation on start, actual screensaver as titlemap etc. Work in progress

![](/screenshot.avif)

Dependency for `genUDMFMap.py`: `omgifol`
