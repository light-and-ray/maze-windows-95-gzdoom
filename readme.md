## Maze Windows 95 UZDoom/GZDoom

This project is a recreation of Maze Windows 95 screensaver in UZDoom/GZDoom engine with all the features of the original screensaver, including maze generation, walls animation on start, actual screensaver as titlemap etc. Work in progress
